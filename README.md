O projeto é referente ao curso de Ciência de Dados da Alura em parceria da Oracle.

O projeto tem com o objetivo de utilizar um banco de dados em .CSV referente a 4 lojas de empresário, cujo gostaria de vender uma das lojas para usar o dinheiro para investir em um novo negócio. Fui contratado para analisar os dados que possuímos sobre as 4 lojas e decidir qual a melhor loja para ser vendida. 

Foi usado os dicionários Pandas e matplotlib. O pandas foi usado na análise de dados enquanto o matplotlib foi usado exclusivamente para a criação dos gráficos no final do projeto. 

No final de tudo, foi decidido que a loja número 4 é a melhor opção para venda.
